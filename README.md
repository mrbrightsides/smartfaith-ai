## ✨ Key Features

### 🤖 Islamic AI Chat (SmartFaith Chat)
Interactive consultation with 7 specialized AI personas:
- **FiqhBot:** Islamic Law & Worship.
- **SirahBot:** History of the Prophet's Journey & Companions.
- **TarikhBot:** Islamic Civilization & World History.
- **NusantaraBot:** Islamic History & Culture in the Archipelago (Indonesia).
- **MuamalahBot:** Business Ethics & Sharia Economics.
- **GenZBot:** Contemporary Muslim lifestyle discussions.
- **MuallafBot:** Basic guidance for new brothers and sisters.

### 📖 Pro Digital Al-Qur'an
- Official Kemenag (Ministry of Religious Affairs) mushaf text.
- Complete Translation & Commentary (Tafsir).
- Murottal audio per verse.
- Fast surah search feature.

### 🕋 Worship Management
- **Real-time Prayer Times:** Accuracy based on user's geographic location.
- **Nearby Mosques:** OpenStreetMap integration to find places of worship around you.
- **24-Hour Murottal:** Non-stop streaming from world-renowned reciters.

### 🛠️ Utilities & Support Tools
- **Khutbah Generator:** Create high-quality sermon drafts in seconds using AI.
- **Zakat Calculator:** Accurate calculation for Wealth Zakat (Income).
- **Hijri Calendar:** Integrated sunnah fasting schedule & Islamic holidays.
- **Dhikr & Supplications:** Collection of daily prayers with a soothing interface.
- **Live Religious TV:** Direct access to Makkah & Madinah Live broadcasts.
- **Hifz Check:** Help with memorization submission using AI-assisted analysis.

## 🛠️ Installation & Development

If you want to run this application locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/username/smartfaith-ai.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Configuration**
   Create a `.env` file in the root folder and add your API key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the application**
   ```bash
   npm run dev
   ```
